# Henson Trust Case Harvester (SCC 2019 SCC 4)

## What it does:
- Fetches real SCC case: "S.A. v. Metro Vancouver Housing Corp., 2019 SCC 4"
- Extracts relevant estate planning rules (Henson Trust context)
- Saves rules to output/extracted_rules.json
- Logs activity to output/log.txt

## To run:
1. Install dependencies:
   pip install -r requirements.txt

2. Run the harvester:
   python main.py
